Overview

About This Documentation

Это объединённая JavaDoc документация для всех backend модулей приложения Messenger Project. Документация генерируется автоматически из исходного кода с использованием Maven Javadoc Plugin и содержит полное описание всех классов, интерфейсов, методов и пакетов каждого модуля.

Project Architecture

Authorization Service

Обработка аутентификации и авторизации пользователей с использованием JWT токенов

Core API Service

Основная бизнес-логика: управление чатами, сообщениями, пользователями и файлами

Gateway

Точка входа для всех клиентских запросов с маршрутизацией к соответствующим сервисам

WebSocket Server

Real-time соединения для мгновенных сообщений и push-уведомлений
